ISPC


Proyecto Renovación Sistema Gestión de BIBLIOTECA


Este sistema en una primer instancia esta pensado para sea administrado por una persona, teniendo la posibilidad de crear un usuario, libros y alquileres de los mismo. El ejemplo se tomo de desarrollo en java. El sistema de biblioteca esta conectado a una base de datos Mysql, con servidores utilizando php y FrontEnd Angular
